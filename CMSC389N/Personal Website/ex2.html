1. Stack and Queue
link!

Within this project I implemented thestack and queue Advanced Data Types (ADT). This included functions such as pop, peek, and push. Both ADTs implemented abstract classes in order to provide complete functionality. With the addition of an inherited exception to secure the notation of the data structures. These ADTs are limited access, meaning you can only access certain objects within the structure.

2. Morse Code Tree
link!

Within this project is the implementation of a morse code translator using a tree. This is another Advanced Data Type (ADT) which is helpful in organizing data. This specific project is able to parse english sentences into morse code, and vice versa. The tree also incorporates recursive functions to search for specific nodes.

3. Doubly Linked List
link!

This project contains the Java implementation of a Doubly Linked List, this is another Advanced Data Type (ADT). This form is more efficient at accessing elements when compared to the stack and queue data structures. This is due to the ease in ability to iterate the data. Both forward and backward traversal is possible when using a doubly linked list.

4. Linked List
link!

This project consists of a Linked list implemented in C. This Advanced Data Type is similar to a doubly linked list, with the exception that there is no backward traversal. This implementation in C includes memory allocation and de-allocation which is needed in C due to the lack of "Garbage Collector" Java contains in it's development kit.

5. NFA and Regex
link!

This project is an example of OCaml, a functional programming language. The purpose of this program is to parse Regular Expressions (Regex) and Non-deterministic Finite Automata (NFA). This implementation doesn't use any finite iteration, rather the functional language allows for an entirely recursive implementation of the project.
